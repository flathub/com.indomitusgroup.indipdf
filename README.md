# indiPDF

A powerful, native PDF editor for Linux.

## Installation
```bash
flatpak install flathub com.indomitusgroup.indipdf
```

## Features

- Annotations: highlight, underline, strikethrough, freehand drawing, shapes, stamps
- Full text editing of existing PDFs
- Form filling with calculated field support
- Page management: reorder, rotate, delete, extract, merge, split
- Multi-tab interface
- Signatures
- Dark/light theme support
- Print support
